## Data Security

### Encryption and Protection
We implement industry-standard security measures:

**Data Encryption:**
- TLS/HTTPS encryption for data in transit
- AES-256 encryption for sensitive data at rest
- Encrypted database storage with Supabase
- Secure key management and rotation

**Access Controls:**
- Multi-factor authentication options
- Role-based access controls
- Session management and timeout
- IP-based access restrictions
- Regular security audits and penetration testing

**Infrastructure Security:**
- SOC 2 Type II compliant hosting
- Regular security updates and patches
- Intrusion detection and monitoring
- Backup and disaster recovery procedures
- Data center physical security measures

### Data Retention
- Account data: Retained while your account is active
- Financial data: Retained for 7 years after account closure (regulatory compliance)
- Usage logs: Retained for 90 days for security purposes
- Support communications: Retained for 3 years
- Marketing data: Retained until you withdraw consent
